# Specifications



## Instruction Set Architecture (ISA)
* 16-bit RISC
* Fixed instruction length


## Architecture
* Harvard style
* Memory mapped I/O
* 8-bit word, 16-bit instructions


## Components
* 64K memory


## Registers
16 8-bit registers

* R0-R11 - general purpose
* R12 - status register
* R13 - stack pointer
* R14 - global data pointer
* R15 - program counter


## Status Flags
Flags found in ST register
* Z - zero flag
* N - negative flag
* O - overflow flag
* C - carry flag
* I - interrupt flag
* X - unused
